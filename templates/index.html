API da CBTECH

Sistema de Gestão de Serviços Técnicos de Informática

Clientes

Gerencie informações dos clientes da CBTECH.

  • GET /api/clientes

    Lista todos os clientes

  • POST /api/clientes

    Criar novo cliente

Assistências

Controle os serviços técnicos realizados.

  • GET /api/assistencias

    Lista todas as assistências

  • POST /api/assistencias

    Registrar nova assistência

Exemplo de Uso

Como utilizar a API com Python:

import requests

# Configuração base
BASE_URL = 'http://localhost:5000/api'

# Listar clientes
response = requests.get(f'{BASE_URL}/clientes')
clientes = response.json()

# Criar cliente
novo_cliente = {
    "nome": "João Silva",
    "email": "joao@example.com",
    "nif": "123456789",
    "telefone": "912345678",
    "morada": "Rua Example, 123"
}
response = requests.post(
    f'{BASE_URL}/clientes', 
    json=novo_cliente
)
